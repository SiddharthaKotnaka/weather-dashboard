# 🌦 Crazy Weather Dashboard

A modern, responsive, and interactive weather dashboard built using **HTML, CSS, and JavaScript**. This application provides real-time weather information, 5-day forecasts, weather analytics, location-based weather detection, interactive temperature charts, dark mode, and search history functionality.

---

## 🚀 Features

### 🌍 Real-Time Weather Information

* Search weather by city name
* Current temperature
* Weather condition
* Humidity
* Wind speed
* Weather icons

### 📍 Location-Based Weather

* Detects user's current location
* Fetches live weather automatically using Geolocation API

### 📊 5-Day Weather Forecast

* Daily forecast cards
* Clickable forecast details
* Temperature prediction for upcoming days

### 📈 Temperature Trend Charts

* Interactive weather charts using Chart.js
* Visual representation of forecast temperature trends
* Hourly temperature analysis

### 🎨 Dynamic Weather Backgrounds

Background changes automatically according to weather conditions:

* Clear Sky
* Clouds
* Rain
* Thunderstorm
* Snow

### 🌙 Dark Mode

* One-click dark mode toggle
* Theme preference saved using Local Storage

### 💾 Search History

* Stores recently searched cities
* Quick access to previous searches

### 🌐 Saved Cities Widget

Displays weather for:

* Hyderabad
* Delhi
* Mumbai
* London
* New York

### 📱 Fully Responsive Design

Optimized for:

* Desktop
* Tablet
* Mobile Devices

## ⚙️ Installation

### 1. Clone Repository

```bash
git clone https://github.com/yourusername/weather-dashboard.git
```

### 2. Open Project

```bash
cd weather-dashboard
```

### 3. Configure API Key

Open `script.js`

Replace:

```javascript
const apiKey = "YOUR_API_KEY";
```

with your OpenWeatherMap API key.

### 4. Run Project

Simply open:

```text
index.html
```

in your browser.

---

## 🔑 Get OpenWeatherMap API Key

1. Create an account at OpenWeatherMap.
2. Generate a free API key.
3. Replace the API key inside `script.js`.
